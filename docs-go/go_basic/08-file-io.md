## –†–∞–±–æ—Ç–∞ —Å JSON

JSON (JavaScript Object Notation) ‚Äî —ç—Ç–æ —Å–∞–º—ã–π –ø–æ–ø—É–ª—è—Ä–Ω—ã–π —Ñ–æ—Ä–º–∞—Ç –¥–ª—è –æ–±–º–µ–Ω–∞ –¥–∞–Ω–Ω—ã–º–∏: –º–µ–∂–¥—É —Å–µ—Ä–≤–µ—Ä–æ–º –∏ –∫–ª–∏–µ–Ω—Ç–æ–º, –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –Ω–∞—Å—Ç—Ä–æ–µ–∫, API –∏ —Ç.–¥.   –í Go —Ä–∞–±–æ—Ç–∞ —Å JSON –≤—Å—Ç—Ä–æ–µ–Ω–∞ –≤ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—É—é –±–∏–±–ª–∏–æ—Ç–µ–∫—É ‚Äî –ø–∞–∫–µ—Ç **`encoding/json`**.

### –û—Å–Ω–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏
- **Marshal** ‚Äî –ø—Ä–µ–≤—Ä–∞—â–∞–µ—Ç Go-—Å—Ç—Ä—É–∫—Ç—É—Ä—ã/–∑–Ω–∞—á–µ–Ω–∏—è –≤ JSON (—Å–µ—Ä–∏–∞–ª–∏–∑–∞—Ü–∏—è, "–∑–∞–ø–∏—Å—å").
- **Unmarshal** ‚Äî –ø—Ä–µ–≤—Ä–∞—â–∞–µ—Ç JSON –≤ Go-—Å—Ç—Ä—É–∫—Ç—É—Ä—ã/–∑–Ω–∞—á–µ–Ω–∏—è (–¥–µ—Å–µ—Ä–∏–∞–ª–∏–∑–∞—Ü–∏—è, "—á—Ç–µ–Ω–∏–µ").

### –°—Ç—Ä—É–∫—Ç—É—Ä–∞ –∏ —Ç–µ–≥–∏ json
–ß—Ç–æ–±—ã Go –∑–Ω–∞–ª, –∫–∞–∫ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤—ã–≤–∞—Ç—å –ø–æ–ª—è —Å—Ç—Ä—É–∫—Ç—É—Ä—ã –≤ JSON, –∏—Å–ø–æ–ª—å–∑—É—é—Ç **—Ç–µ–≥–∏** (backticks):

```go

    Email string `json:"email"`
    Password string `json:"-"`        // "-" ‚Äî –∏–≥–Ω–æ—Ä–∏—Ä–æ–≤–∞—Ç—å –ø–æ–ª–µ –ø–æ–ª–Ω–æ—Å—Ç—å—é
    Admin bool `json:"is_admin,omitempty"` // omitempty ‚Äî –Ω–µ –≤—ã–≤–æ–¥–∏—Ç—å, –µ—Å–ª–∏ false –∏–ª–∏ –ø—É—Å—Ç–æ–µ
}
```

–í–∞–∂–Ω—ã–µ –æ–ø—Ü–∏–∏ —Ç–µ–≥–æ–≤:
- `json:"name"` ‚Äî –∏–º—è –ø–æ–ª—è –≤ JSON
- `json:"-"` ‚Äî –ø—Ä–æ–ø—É—Å—Ç–∏—Ç—å –ø–æ–ª–µ
- `json:",omitempty"` ‚Äî –Ω–µ –≤–∫–ª—é—á–∞—Ç—å –ø–æ–ª–µ, –µ—Å–ª–∏ –æ–Ω–æ –ø—É—Å—Ç–æ–µ ("" / 0 / false / nil)

## –†–∞–±–æ—Ç–∞ —Å CSV 

CSV (Comma-Separated Values) ‚Äî —ç—Ç–æ –ø—Ä–æ—Å—Ç–æ–π —Ç–µ–∫—Å—Ç–æ–≤—ã–π —Ñ–æ—Ä–º–∞—Ç –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è —Ç–∞–±–ª–∏—á–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö: —Å—Ç—Ä–æ–∫–∏ ‚Äî —ç—Ç–æ –∑–∞–ø–∏—Å–∏, —Å—Ç–æ–ª–±—Ü—ã —Ä–∞–∑–¥–µ–ª–µ–Ω—ã –∑–∞–ø—è—Ç—ã–º–∏ (–∏–ª–∏ –¥—Ä—É–≥–∏–º —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª–µ–º). –û—á–µ–Ω—å –ø–æ–ø—É–ª—è—Ä–µ–Ω –¥–ª—è —ç–∫—Å–ø–æ—Ä—Ç–∞/–∏–º–ø–æ—Ä—Ç–∞ –¥–∞–Ω–Ω—ã—Ö (Excel, Google Sheets, –±–∞–∑—ã –¥–∞–Ω–Ω—ã—Ö –∏ —Ç.–¥.).
