## –ò–Ω—Ç–µ—Ä—Ñ–µ–π—Å—ã

**interface *(–ò–Ω—Ç–µ—Ä—Ñ–µ–π—Å)*** –≤ Go ‚Äî —ç—Ç–æ –∫–∞–∫ —Å–ø–∏—Å–æ–∫ —É–º–µ–Ω–∏–π: —Ç—ã –≥–æ–≤–æ—Ä–∏—à—å ¬´–º–Ω–µ –Ω—É–∂–Ω–∞ –≤–µ—â—å, –∫–æ—Ç–æ—Ä–∞—è —É–º–µ–µ—Ç –¥–µ–ª–∞—Ç—å –≤–æ—Ç —ç—Ç–æ¬ª, –∏ –Ω–µ –≤–∞–∂–Ω–æ, –∫—Ç–æ –æ–Ω–∞ —Ç–∞–∫–∞—è. 

–ù–∞–ø—Ä–∏–º–µ—Ä, –æ–±—ä—è–≤–ª—è–µ—à—å `type Speaker interface { Speak() string }` ‚Äî –∑–Ω–∞—á–∏—Ç, –Ω—É–∂–µ–Ω –∫—Ç–æ-—Ç–æ —Å –º–µ—Ç–æ–¥–æ–º `Speak()`, –∫–æ—Ç–æ—Ä—ã–π –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å—Ç—Ä–æ–∫—É. –õ—é–±–∞—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ (—Å–æ–±–∞–∫–∞, –∫–æ—à–∫–∞, —á–µ–ª–æ–≤–µ–∫), —É –∫–æ—Ç–æ—Ä–æ–π –µ—Å—Ç—å —Ç–∞–∫–æ–π –º–µ—Ç–æ–¥, –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –ø–æ–¥—Ö–æ–¥–∏—Ç –ø–æ–¥ —ç—Ç–æ—Ç –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å ‚Äî –Ω–∏—á–µ–≥–æ –ª–∏—à–Ω–µ–≥–æ –ø–∏—Å–∞—Ç—å –Ω–µ –Ω–∞–¥–æ! –ü–∏—à–µ—à—å —Ñ—É–Ω–∫—Ü–∏—é `func greet(s Speaker) { fmt.Println(s.Speak()) }` ‚Äî –∏ –æ–Ω–∞ —Ä–∞–±–æ—Ç–∞–µ—Ç —Å —Å–æ–±–∞–∫–æ–π (¬´–ì–∞–≤!¬ª), –∫–æ—à–∫–æ–π (¬´–ú—è—É!¬ª) –∏–ª–∏ –∫–µ–º —É–≥–æ–¥–Ω–æ –µ—â—ë, —É –∫–æ–≥–æ –µ—Å—Ç—å —ç—Ç–æ —É–º–µ–Ω–∏–µ. –≠—Ç–æ –¥–µ–ª–∞–µ—Ç –∫–æ–¥ –≥–∏–±–∫–∏–º: –æ–¥–Ω–∞ —Ñ—É–Ω–∫—Ü–∏—è ‚Äî –º–Ω–æ–≥–æ —Ä–∞–∑–Ω—ã—Ö —Ç–∏–ø–æ–≤. 

–ò–Ω—Ç–µ—Ä—Ñ–µ–π—Å—ã ‚Äî –ø—Ä–æ—Å—Ç–æ–π –∏ –º–æ—â–Ω—ã–π —Å–ø–æ—Å–æ–± —Å–∫–∞–∑–∞—Ç—å *¬´–º–Ω–µ –≤—Å—ë —Ä–∞–≤–Ω–æ, –∫—Ç–æ —Ç—ã, –≥–ª–∞–≤–Ω–æ–µ ‚Äî —á—Ç–æ —Ç—ã —É–º–µ–µ—à—å¬ª*

```go
type Shape interface {
    Area() float64
    Perimeter() float64
}

type Circle struct{ Radius float64 }
type Rectangle struct{ Width, Height float64 }

func (c Circle) Area() float64      { return 3.14159 * c.Radius * c.Radius }

func (r Rectangle) Area() float64      { return r.Width * r.Height }
func (r Rectangle) Perimeter() float64 { return 2 * (r.Width + r.Height) }
```

### Type switch 

**Type switch** ‚Äî —ç—Ç–æ —Å–ø–µ—Ü–∏–∞–ª—å–Ω–∞—è –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è `switch` –≤ Go, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø—Ä–æ–≤–µ—Ä–∏—Ç—å **—Ç–∏–ø** –∑–Ω–∞—á–µ–Ω–∏—è –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–∞ (`interface{}`) –∏ –≤—ã–ø–æ–ª–Ω–∏—Ç—å —Ä–∞–∑–Ω—ã–µ –¥–µ–π—Å—Ç–≤–∏—è –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Ä–µ–∞–ª—å–Ω–æ–≥–æ —Ç–∏–ø–∞ "–ø–æ–¥ –∫–∞–ø–æ—Ç–æ–º".

–≠—Ç–æ –æ–¥–∏–Ω –∏–∑ —Å–ø–æ—Å–æ–±–æ–≤ —Ä–∞–±–æ—Ç—ã —Å **–¥–∏–Ω–∞–º–∏—á–µ—Å–∫–∏–º–∏ —Ç–∏–ø–∞–º–∏** –≤ —Å—Ç–∞—Ç–∏—á–µ—Å–∫–∏ —Ç–∏–ø–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–æ–º —è–∑—ã–∫–µ Go. –û–Ω –Ω—É–∂–µ–Ω, –∫–æ–≥–¥–∞ —É —Ç–µ–±—è –µ—Å—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ —Ç–∏–ø–∞ `interface{}`, –∏ —Ç—ã —Ö–æ—á–µ—à—å —É–∑–Ω–∞—Ç—å, —á—Ç–æ –∏–º–µ–Ω–Ω–æ –≤ –Ω—ë–º –ª–µ–∂–∏—Ç: int, string, —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –∏–ª–∏ —á—Ç–æ-—Ç–æ –µ—â—ë.

#### –°–∏–Ω—Ç–∞–∫—Å–∏—Å
```go

    fmt.Println("–ù–µ–∏–∑–≤–µ—Å—Ç–Ω—ã–π —Ç–∏–ø:", reflect.TypeOf(value))
```

- `value.(type)` ‚Äî —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å, –¥–æ—Å—Ç—É–ø–Ω—ã–π **—Ç–æ–ª—å–∫–æ –≤–Ω—É—Ç—Ä–∏ switch**.
- `v` ‚Äî —ç—Ç–æ –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è, –∫–æ—Ç–æ—Ä–∞—è –±—É–¥–µ—Ç –∏–º–µ—Ç—å –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–π —Ç–∏–ø –≤ –∫–∞–∂–¥–æ–º `case`.
