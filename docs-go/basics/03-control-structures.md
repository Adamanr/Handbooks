#### 1. **–û–±–ª–∞—Å—Ç—å –≤–∏–¥–∏–º–æ—Å—Ç–∏ (scope) –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π `v` ‚Äî –ì–ª–∞–≤–Ω–æ–µ –≤–∏–¥–∏–º–æ–µ —Ä–∞–∑–ª–∏—á–∏–µ**

#### 3. **–ò–¥–∏–æ–º–∞—Ç–∏—á–Ω–æ—Å—Ç—å –∏ —Å—Ç–∏–ª—å Go**
–í–æ—Ç –≥–¥–µ **–Ω–∞—Å—Ç–æ—è—â–∞—è —Ä–∞–∑–Ω–∏—Ü–∞** –≤ –ø—Ä–∞–∫—Ç–∏–∫–µ:

### –û–ø–µ—Ä–∞—Ç–æ—Ä `switch`

–í Go –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è `switch` –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω–∞ –¥–ª—è –≤—ã–±–æ—Ä–∞ –æ–¥–Ω–æ–≥–æ –∏–∑ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ –≤–∞—Ä–∏–∞–Ω—Ç–æ–≤ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –∫–æ–¥–∞ –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –∑–Ω–∞—á–µ–Ω–∏—è –≤—ã—Ä–∞–∂–µ–Ω–∏—è (–∏–ª–∏ —É—Å–ª–æ–≤–∏–π –≤ –æ—Å–æ–±–æ–º —Ñ–æ—Ä–º–∞—Ç–µ) ‚Äî —ç—Ç–æ —É–¥–æ–±–Ω–∞—è –∏ —á–∏—Ç–∞–µ–º–∞—è –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–∞ –¥–ª–∏–Ω–Ω–æ–π —Ü–µ–ø–æ—á–∫–µ `if-else if-else`.

–í—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è —Ç–æ–ª—å–∫–æ –ø–µ—Ä–≤—ã–π –ø–æ–¥—Ö–æ–¥—è—â–∏–π `case` (–∏–ª–∏ `default`, –µ—Å–ª–∏ –Ω–∏—á–µ–≥–æ –Ω–µ –ø–æ–¥–æ—à–ª–æ), –∏ –±–ª–∞–≥–æ–¥–∞—Ä—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–æ–º—É `break` –≤ –∫–æ–Ω—Ü–µ –∫–∞–∂–¥–æ–≥–æ `case` (–≤ –æ—Ç–ª–∏—á–∏–µ –æ—Ç C) –∫–æ–¥ –Ω–µ ¬´–ø—Ä–æ–≤–∞–ª–∏–≤–∞–µ—Ç—Å—è¬ª –¥–∞–ª—å—à–µ ‚Äî –µ—Å–ª–∏ –Ω—É–∂–Ω–æ –ø—Ä–æ–¥–æ–ª–∂–∏—Ç—å –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ —Å–ª–µ–¥—É—é—â–µ–≥–æ `case`, –¥–æ–±–∞–≤–ª—è—é—Ç —Å–ø–µ—Ü–∏–∞–ª—å–Ω–æ–µ —Å–ª–æ–≤–æ `fallthrough`. `switch` —á–∞—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑—É—é—Ç –¥–ª—è –æ–±—Ä–∞–±–æ—Ç–∫–∏ —Ä–∞–∑–Ω—ã—Ö –∑–Ω–∞—á–µ–Ω–∏–π –æ–¥–Ω–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π (—Ç–∏–ø, —Å—Ç–∞—Ç—É—Å, –∫–æ–¥ –æ—à–∏–±–∫–∏, –¥–µ–Ω—å –Ω–µ–¥–µ–ª–∏), —Ä–∞–∑–±–æ—Ä–∞ —Ç–µ–≥–æ–≤ –∏–ª–∏ –¥–∞–∂–µ –¥–ª—è –∑–∞–º–µ–Ω—ã —Å–ª–æ–∂–Ω—ã—Ö —É—Å–ª–æ–≤–Ω—ã—Ö —Ü–µ–ø–æ—á–µ–∫, –¥–µ–ª–∞—è –∫–æ–¥ —á–∏—â–µ –∏ –±—ã—Å—Ç—Ä–µ–µ (–∫–æ–º–ø–∏–ª—è—Ç–æ—Ä –æ–ø—Ç–∏–º–∏–∑–∏—Ä—É–µ—Ç –ø—Ä–æ—Å—Ç—ã–µ `switch` –ª—É—á—à–µ, —á–µ–º `if-else`).

## –¶–∏–∫–ª—ã 

–í Go –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è `for` ‚Äî —ç—Ç–æ **–µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω—ã–π** —Ü–∏–∫–ª –≤ —è–∑—ã–∫–µ (–Ω–µ—Ç while, do-while –∏ —Ç.–ø.), –Ω–æ –æ–Ω –Ω–∞—Å—Ç–æ–ª—å–∫–æ –≥–∏–±–∫–∏–π, —á—Ç–æ –ø–æ–∫—Ä—ã–≤–∞–µ—Ç –≤—Å–µ –≤–æ–∑–º–æ–∂–Ω—ã–µ —Å—Ü–µ–Ω–∞—Ä–∏–∏. –ö–ª–∞—Å—Å–∏—á–µ—Å–∫–∏–π –≤–∏–¥ ‚Äî –∫–∞–∫ –≤ C: `for i := 0; i < 10; i++ { ... }`, –≥–¥–µ –µ—Å—Ç—å –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏—è, —É—Å–ª–æ–≤–∏–µ –∏ –ø–æ—Å—Ç-–∏—Ç–µ—Ä–∞—Ü–∏—è. 

–û–¥–Ω–∞–∫–æ –≤ Go `for` –º–æ–∂–µ—Ç –±—ã—Ç—å –∏ –≤ —É–ø—Ä–æ—â—ë–Ω–Ω—ã—Ö —Ñ–æ—Ä–º–∞—Ö: –ø—Ä–æ—Å—Ç–æ —Å —É—Å–ª–æ–≤–∏–µ–º ‚Äî `for i < 10 { i++ ... }` (–∞–Ω–∞–ª–æ–≥ while), –∏–ª–∏ –≤–æ–≤—Å–µ –±–µ–∑ –Ω–∏—á–µ–≥–æ ‚Äî `for { ... }` (–±–µ—Å–∫–æ–Ω–µ—á–Ω—ã–π —Ü–∏–∫–ª, –ø—Ä–µ—Ä—ã–≤–∞–µ—Ç—Å—è `break` –∏–ª–∏ `return`). –°–∞–º–∞—è –ø–æ–ø—É–ª—è—Ä–Ω–∞—è –∏–¥–∏–æ–º–∞ ‚Äî `for index, value := range collection { ... }`, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç —É–¥–æ–±–Ω–æ –ø–µ—Ä–µ–±–∏—Ä–∞—Ç—å –º–∞—Å—Å–∏–≤—ã, —Å—Ä–µ–∑—ã, —Å—Ç—Ä–æ–∫–∏ (–ø–æ —Ä—É–Ω–∞–º), –º–∞–ø—ã (–∫–ª—é—á–∏ –∏ –∑–Ω–∞—á–µ–Ω–∏—è) –∏ –∫–∞–Ω–∞–ª—ã. 

–í–Ω—É—Ç—Ä–∏ —Ü–∏–∫–ª–∞ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `break` (–≤—ã—Ö–æ–¥ –∏–∑ —Ü–∏–∫–ª–∞), `continue` (–ø–µ—Ä–µ—Ö–æ–¥ –∫ —Å–ª–µ–¥—É—é—â–µ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏) –∏ –¥–∞–∂–µ –º–µ—Ç–∫–∏ (labels) –¥–ª—è –≤—ã—Ö–æ–¥–∞ –∏–∑ –≤–ª–æ–∂–µ–Ω–Ω—ã—Ö —Ü–∏–∫–ª–æ–≤ ‚Äî –Ω–∞–ø—Ä–∏–º–µ—Ä, `outer: for ... { for ... { if condition { break outer } } }`. 

`for` –≤ Go –ø—Ä–æ—Å—Ç, –ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º, —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–µ–Ω –∏ –∏–¥–µ–∞–ª—å–Ω–æ –ø–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è –≤—Å–µ–≥–æ: –æ—Ç –ø—Ä–æ—Å—Ç–æ–≥–æ —Å—á—ë—Ç–∞ –¥–æ –æ–±—Ä–∞–±–æ—Ç–∫–∏ –¥–∞–Ω–Ω—ã—Ö –≤ –∫–æ–ª–ª–µ–∫—Ü–∏—è—Ö, –¥–µ–ª–∞—è –∫–æ–¥ —á–∏—Å—Ç—ã–º –∏ —á–∏—Ç–∞–µ–º—ã–º.

### –ö–ª–∞—Å—Å–∏—á–µ—Å–∫–∏–µ —Ñ–æ—Ä–º—ã `for`

### –£–ø—Ä–∞–≤–ª–µ–Ω–∏–µ —Ü–∏–∫–ª–∞–º–∏

- `break` ‚Äî –≤—ã–π—Ç–∏ –∏–∑ —Ü–∏–∫–ª–∞ –ø–æ–ª–Ω–æ—Å—Ç—å—é
- `continue` ‚Äî –ø—Ä–æ–ø—É—Å—Ç–∏—Ç—å –æ—Å—Ç–∞—Ç–æ–∫ —Ç–µ–∫—É—â–µ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏ –∏ –ø–µ—Ä–µ–π—Ç–∏ –∫ —Å–ª–µ–¥—É—é—â–µ–π

```go
        fmt.Printf("(%d,%d) ", i, j)

    fmt.Println()

/*  –í—ã–≤–æ–¥:
    (0,0) (0,1) (0,2) (0,3) (0,4)

    (2,0) (2,1) (2,2)

```
